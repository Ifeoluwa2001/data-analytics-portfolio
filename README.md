## 📁 Projects in this repo

### 1. `Sobowale_Ifeoluwa_TS_Academy_Excel_Project.xlsx`
**Sales Performance Dashboard (Excel)**
Interactive sales dashboard built with pivot tables and slicers, tracking revenue, cost, profit, and customer trends by region and category. Total Revenue: ₦2.33B · Profit: ₦465.7M · Customers: 2,098.
**Tools:** Excel, Pivot Tables, Slicers

### 2. `Sobowale_Ifeoluwa_Power_BI_Project.pbix`
**Sales Performance Dashboard (Power BI)**
Power BI rebuild of the sales analysis, segmented by customer income level, product color, and brand. Revenue: $3M · Profit: $932K · Customers: 250.
**Tools:** Power BI, DAX, Data Modeling

### 3. `Sobowale_Ifeoluwa_TS_Academy_Correct_Capstone_Project.pbix`
**End-to-End Logistics Operations & Performance Analysis (Capstone)**
A 5-page Power BI dashboard built across a 14-table logistics dataset — fleet, fuel, drivers, claims, and routes. Total Revenue: $298.6M · Profit Margin: 65.18% · Total Trips: 85K.
**Tools:** Power BI, Power Query (ETL), DAX, Relational Data Modeling

### 4. `DECODELBS__PROJECT_1.xlsx`
**Data Cleaning Report — Transaction Dataset**
Cleaned and standardized 1,200 rows of raw transaction data across 14 columns. Fixed null coupon codes, verified zero duplicate Order IDs, standardized text casing, currency formatting, and ISO 8601 date formats. Achieved a 0% error rate on unique identifiers, dates, and number formats.
**Tools:** Excel, Data Cleaning, Data Profiling

### 5. `PROJECT_2_DECODELABS.xlsx`
**Sales Data Analysis & Trend Report**
Statistical analysis of 1,200 orders totaling $1.26M in revenue. Surfaces key insights: declining year-over-year revenue (2023 → 2025), top revenue drivers (Chairs, Printers), payment method trends, and a flagged logistics issue — cancelled and returned orders combined outnumber successful deliveries.
**Tools:** Excel, Statistical Analysis, Trend Analysis

- **`.xlsx`** files open directly in Excel or Google Sheets.
- **`.pbix`** files require [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free) to open and interact with.

GitHub doesn't preview either format inline — click into each file and use the **Download** button to get the working version.
